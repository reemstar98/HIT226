To the naked eye the basic colours used to create this website as can be seen are:

  • ➢ Grey
  • ➢ Light Grey
  • ➢ Black
  • ➢ White
  • ➢ Dark Blue
  • ➢ Light or Sky blue

However in HTML colour names are much more specific. Each colour has a specific name and a specific code needed to utalize the colour. Along with this many things need to be taken into account whe choosing a websites colour scheme. A few questions to ask yourself when choosing a colour scheme for your website are:

  • ➢ Can the user read without straining their eyes?
  • ➢ What if a person with colour blindness was to use the site? Would they find it difficult?
  • ➢ Does the colour contraster pass this mix of forground and background colours?
  • ➢ What is this website being made for?
  • ➢ Does the client want any specification regarding the colour?
  • ➢ Will this website colour scheme be able to work on every page?

The goal for this websites colour palette was to pass AAA for all background and forground colour combinations. This was aicheved for both normal and bold text for five out of six combinations. The only combination that did not aicheve this AAA level pass was the website navigation colour when active. However although it did not pass AAA in normal text, it did pass AAA in bold text and as this website only used the combination in bold text it has been considered in this case a pass overall.(WebAim 2017)
