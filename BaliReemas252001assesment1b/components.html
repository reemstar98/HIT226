Buttons

The <button> tag can be used in html to create a button on a webpage. This button can be linked to a message, page or picture depending on preferance. It can also act as a tool to do things such as close browsers and minimize depending on the type of code used to create it. (w3schools 2017)

Contact Chips

John Doe

Chips represent complex entities in small blocks. Contact chips are commonly seen today on mobile phones. They are used to display phone contact on various apps. The look of contact chips can vary depending on the operating system. Chips can also be found in many emailing systems. When chosing a contact to send a file to contact chips can be used for easy readability of the emailer.

A table allows the user to arrange data, meta data and text into rows and collums. The table is often assosciated with the caption tag. A table should not be used as a way to create an entire webpage layout as that will lead to problems in the long run. What the table represents in terms of data is up to the creator of the table.
